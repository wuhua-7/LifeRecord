# 🌟 生活記錄系統

簡單優雅的每日生活追蹤工具，幫助你記錄和管理日常生活。

## ✨ 功能特色

- 📝 每日記錄：記錄三餐、運動、睡眠等生活細節
- 😊 心情追蹤：記錄每日心情狀態
- ⭐ 評分系統：為每一天打分，追蹤生活品質
- 📊 數據統計：查看總記錄數和平均評分
- 💾 本地儲存：使用 localStorage 保存數據
- 📱 響應式設計：完美支援手機和桌面裝置

## 🚀 快速開始

### 本地運行

```bash
# 安裝依賴（可選）
npm install

# 啟動開發伺服器
npm run dev

# 或直接打開 public/index.html
```

### 部署到 Vercel

1. Fork 此專案
2. 在 Vercel 導入專案
3. 點擊部署即可

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

## 🛠️ 技術棧

- **React 18** - UI 框架
- **Tailwind CSS** - 樣式框架
- **DaisyUI** - UI 組件庫
- **LocalStorage** - 數據持久化

## 📝 使用說明

1. 填寫姓名和日期
2. 記錄三餐內容
3. 記錄運動和睡眠時間
4. 選擇當日心情
5. 為這一天評分（1-10）
6. 添加額外備註
7. 點擊提交保存記錄
